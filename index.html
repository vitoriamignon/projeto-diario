Terçou com muito aprendizado - Terminal

01 de Julho de 2025.

Começamos hoje a estudar sobre Terminal, o que é, como encontrar, sistema de arquivos e caminhos de arquivos.

Uma ferramenta poderosa para desenvolvedores, proporcionando rapidez, automação e controle total sobre o sistema!

Na parte prática aprendemos alguns comandos essenciais, como:

  • pwd: Mostra o diretório atual;
  • cd: Navegação entre pastas;
  • mkdir: Criação de novas pastas;
  • touch: Criação de arquivos vazios;
  • cp, mv, rm, rmdir: Utilizados para copiar, mover, renomear e remover arquivos diretórios.

⚠ com comando: "rm -r" - Remove recursivamente (conteudo interno também) ⚠

Pensamentos para hoje:

Quartou com Git e GitHub

02 de Julho de 2025.

Na aula de hoje, compreendemos o uso do Git e GitHub no desenvolvimento de projetos e suas diferenças.

  • Git: É uma ferramenta instalada localmente no computador, responsável por rastrear alterações nos arquivos.
  • GitHub: É uma plataforma online onde os repositórios Git podem ser amarzenados, compartilhados e colaborados com outras pessoas.

Pensamentos para hoje:

Acredite em si mesmo, você é capaz de grandes coisas!
